LLM

コマンド 説明 パラメータ
/chat AIとメンションなしで対話します。 message, image_url (任意)
/set-ai-bio このチャンネルのAIの性格/役割(bio)を設定します。 bio
/show-ai-bio このチャンネルのAIの現在のbioを表示します。 -
/reset-ai-bio このチャンネルのAIのbioをデフォルトに戻します。 -
/set-user-bio AIにあなたの情報を記憶させます。 bio, mode ('上書き' or '追記')
/show-user-bio AIが記憶しているあなたの情報を表示します。 -
/reset-user-bio AIが記憶しているあなたの情報をすべて削除します。 -
/memory-save グローバル共有メモリに情報を保存します。 key, value
/memory-list グローバル共有メモリの情報を一覧表示します。 -
/memory-delete グローバル共有メモリから情報を削除します。 key
/switch-models このチャンネルで使用するAIモデルを切り替えます。 model
/switch-models-default-server このチャンネルのAIモデルをサーバーのデフォルト設定に戻します。 -
/switch-image-model このチャンネルの画像生成モデルを切り替えます。 model
/reset-image-model このチャンネルの画像生成モデルをデフォルトに戻します。 -
/show-image-model このチャンネルの現在の画像生成モデルを表示します。 -
/list-image-models 利用可能な画像生成モデルの一覧を表示します。 provider (任意)
/llm_help LLM (AI対話) 機能のヘルプと利用ガイドラインを表示します。 -
/clear_history 現在の会話スレッドの履歴をクリアします。 -

音楽

コマンド 説明 パラメータ
/play 曲を再生またはキューに追加します。 query
/seek 再生位置を指定した時刻に移動します。 time
/pause 再生を一時停止します。 -
/resume 一時停止中の再生を再開します。 -
/skip 再生中の曲をスキップします。 -
/stop 再生を停止し、キューをクリアします。 -
/leave ボットをボイスチャンネルから切断します。 -
/queue 現在の再生キューを表示します。 -
/nowplaying 現在再生中の曲の情報を表示します。 -
/shuffle 再生キューをシャッフルします。 -
/clear 再生キューを空にします。 -
/remove キューから指定した番号の曲を削除します。 index
/volume 音量を変更します (0-200)。 level
/loop ループ再生モードを設定します。 mode ('off', 'one', or 'all')
/join ボットをあなたのいるボイスチャンネルに接続します。 -
/music_help 音楽機能のコマンド一覧と使い方を表示します。 -
/reload music_cog 音楽Cogを再読み込みして、問題をリセットします。 -

TTS

コマンド 説明 パラメータ
/tts volume TTSの音量を設定します (0-200%)。 volume
/speech enable このチャンネルのメッセージ読み上げを有効にします。 -
/speech disable メッセージ読み上げを無効にします。 -
/speech skip 現在の読み上げをスキップします。 -
/autojoin enable あなたがVCに参加した際、BOTも自動で参加するようにします。 -
/autojoin disable BOTの自動参加設定を解除します。 -
/join-leave-notification enable VCへの入退室を音声で通知するようにします。 -
/join-leave-notification disable VCへの入退室通知を無効にします。 -
/say テキストを音声で読み上げます。 text, model_id (任意), style (任意), style_weight (任意), speed (任意)
/dictionary add 読み上げ辞書に単語を追加します。 word, reading
/dictionary remove 読み上げ辞書から単語を削除します。 word
/dictionary list 登録されている辞書の一覧を表示します。 -
/dictionary search 辞書から単語を検索します。 query

/updates Botの最新のアップデート履歴（コミットログ）を表示します。 -

/help Botのヘルプ情報を表示します。 -
